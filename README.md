# OSCify | VRChat to Spotify Controller
OSCify is a small Python program that lets you control Spotify Desktop on Windows from your VRChat avatar using it's parameters.

### Avatar Controls
Play / Pause
Next Track
Previous Track
Volume Control

### VRChat Parameter Setup
| Parameter Name | Parameter Type | Menu Type |
| ------------- | ------------- | ------------- |
| skipTrack  | Bool | Button  |
| previousTrack  | Bool  | Button  |
| pausePlay  | Bool  | Button  |
| spotVolume  | Float  | Radial Puppet  |

VRChat menu icons provided by [Flaticon](https://www.flaticon.com)
